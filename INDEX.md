## üìû Quick Reference

### Start App
```powershell
docker-compose up --build
```

### Stop App
```powershell
docker-compose down
```

### View Logs
```powershell
docker-compose logs -f
```

### Clean Reset
```powershell
docker-compose down -v
docker-compose up --build
```

### Access URLs
- Frontend: http://localhost
- Backend: http://localhost:5000
- API Docs: [API_DOCUMENTATION.md](API_DOCUMENTATION.md)
